# Portfolio Website

This is a basic portfolio website built with HTML, CSS, and JavaScript. It includes sections for About, Projects, and Contact. Customize the content and styles as needed.

## Structure
- `index.html`: Main homepage
- `styles.css`: Stylesheet for the site
- `script.js`: JavaScript for interactivity
- `assets/`: Folder for images and other assets

## Usage
Open `index.html` in your browser to view the site.

## Customization
Replace placeholder text and images with your own information and assets.
